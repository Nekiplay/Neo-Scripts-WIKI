# Catboost

## `loadModel(path)`

Load model from path

**Parameters:**

* `path` (string).

**Return:**&#x20;

* [model](cat-boost-model/)

**Example Usage:**

```lua
-- Example code showing how to use the function
local catboost = require("catboost")
local model = catboost.loadModel("model.cbm")
```
